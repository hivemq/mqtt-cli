| Option | Long Version       | Explanation                                                                         | Default                 |
|--------|--------------------|-------------------------------------------------------------------------------------|-------------------------|
| `-u`   | `--url`            | The URL of the HiveMQ REST API endpoint.                                            | `http://localhost:8888` |
| `-r`   | `--rate`           | The rate limit of the rest calls to the HiveMQ API endpoint in requests per second. | 1500                    |
